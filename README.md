# Brain-Controlled Drone
## Overview
This project uses electroencephalogram (EEG) technology to track brain signals. We uses these brain signals to send commands to the Tello Drone to control it.

## Equipment Used
* EMOTIV EPOCHx headset
* Emotiv Cortex API
* Tello Drone
* Python
* DJITello library
